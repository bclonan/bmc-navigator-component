# Spacing System

The Your Town Finance spacing system provides consistent rhythm and visual hierarchy across all financial application interfaces. Our spacing scale is based on a 4px base unit to ensure precise alignment and optimal usability.

## Base Unit System

**Base Unit**: 4px
**Scale**: Multiples of 4px for consistent spacing relationships

## Usage Guidelines

### Consistency Rules
1. **Always use multiples of 4px** for spacing values
2. **Maintain visual rhythm** by using consistent spacing patterns
3. **Scale spacing with component size** - larger components get more padding
4. **Group related elements** with smaller spacing, separate unrelated elements with larger spacing

### Financial Application Context
- **Form sections**: 24px spacing between major sections
- **Input fields**: 16px spacing between fields, 8px for label-to-input
- **Cards and panels**: 24px internal padding for comfortable reading
- **Action buttons**: 16px gap between related actions
- **Navigation elements**: 16px gap for optimal touch targets

### Responsive Considerations
- **Mobile**: Reduce spacing by 25% for compact layouts
- **Tablet**: Use standard spacing values
- **Desktop**: Use standard or increase by 25% for larger screens

### Accessibility
- **Touch targets**: Minimum 44px for interactive elements
- **Reading comfort**: 24px line height for optimal readability
- **Visual separation**: Sufficient spacing to distinguish between sections

This spacing system ensures consistent, professional layouts that enhance the user experience in financial applications while maintaining Your Town Finance's brand standards.
