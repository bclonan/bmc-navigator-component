# Typography System

The Your Town Finance typography system uses the Muli font family to create clear, professional communication that builds trust with customers. Our type scale provides hierarchy while maintaining readability across all financial application interfaces.

## Font Family

**Primary**: Muli - A clean, modern sans-serif designed for digital interfaces
**Secondary**: MuliBold - Bold weight variant for emphasis and headings

### Font Loading
```css
@import url('https://fonts.googleapis.com/css2?family=Muli:wght@400;500;600;700&display=swap');
```

## Usage Guidelines

### Hierarchy
1. Use H1 for page titles and primary headlines
2. Use H2-H6 for section organization and content hierarchy
3. Use Body 1 for main content and descriptions
4. Use Body 2 for secondary information and disclaimers

### Financial Content
- **Monetary amounts**: Use bold weights and primary colors
- **Rates and percentages**: Emphasize with color coding (green for good rates)
- **Status updates**: Use appropriate semantic colors
- **Legal text**: Use smaller, lighter weights with sufficient contrast

### Accessibility
- Maintain 4.5:1 contrast ratio for normal text
- Use 3:1 contrast ratio for large text (18pt+ or 14pt+ bold)
- Ensure readable line spacing (1.4-1.6 for body text)
- Provide sufficient color contrast for all text elements

### Implementation
Typography styles are available through CSS custom properties and utility classes for consistent application across all components.
