## Usage Guidelines

### Primary Brand Colors
- **Matt Blue** (`rgb(56, 96, 190)`) - Primary buttons, links, and key UI elements
- **Tardis Blue** (`rgb(1, 52, 116)`) - Secondary actions, headers, and supporting elements

### Semantic Colors
- **Herbal Green** - Loan approvals, successful transactions, positive status indicators
- **Warning Yellow** - Payment due notices, important alerts, caution states
- **Error Red** - Validation errors, failed transactions, critical alerts
- **Info Blue** - General information, neutral notifications, help content

### Accessibility Standards
All color combinations meet WCAG 2.1 AA standards with proper contrast ratios for financial applications.

### Implementation
Colors are available as CSS custom properties for consistent usage across components.
